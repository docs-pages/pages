模态框

eastspire2026/7/19小于 1 分钟euvuirustwasmcomponent-library

euv_modal

模态对话框组件，包含 overlay 遮罩层、标题栏、关闭按钮和内容区。

Props

字段类型默认值说明
title&'static str""模态框标题
onclickOption<Rc<dyn Fn(Event)>>None关闭回调（overlay 和关闭按钮触发）

基本用法

use euv_ui::*;

let modal_open: Signal<bool> = App::use_signal(|| false);

// 切换模态框
let toggle = move |_: Event| {
    let current: bool = modal_open.get();
    modal_open.set(!current);
};

html! {
    euv_button {
        variant: EuvButtonVariant::Primary
        label: "Open Modal"
        onclick: toggle
    }
    if { modal_open.get() } {
        euv_modal {
            title: "Modal Title"
            onclick: toggle
            p { "Modal content goes here." }
            euv_button {
                label: "Close"
                onclick: toggle
            }
        }
    }
}

提示

onclick 同时绑定在 overlay 背景和关闭按钮（×）上，点击任意位置均可关闭。模态框内容通过 children 传入，euv_modal 内部不处理动画，由外层通过 if 条件控制显隐。
