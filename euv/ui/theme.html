主题管理

eastspire2026/7/5大约 1 分钟euvuirustwasmcomponent-library

ThemeState

主题管理工具，提供亮/暗主题切换、系统主题自动检测和根元素 CSS 类管理。

Struct

pub struct ThemeState {
    pub theme: Signal<String>,       // 当前主题名 "light" / "dark"
    pub root_class: Signal<String>,   // 根元素 CSS 类名（含主题类）
}

初始化主题状态

use euv_ui::*;

let mobile_signal: Signal<bool> = App::use_signal(|| false);
let theme_state: ThemeState = ThemeState::use_theme_state(mobile_signal);

use_theme_state 执行以下操作：

  1. 检测系统 prefers-color-scheme 偏好，初始化 theme 信号
  2. 监听系统主题变化，实时更新
  3. 当系统主题或 mobile_signal 变化时，自动组合并更新 root_class 信号

切换主题

// 切换亮/暗主题
let toggle_theme: Option<Rc<dyn Fn(Event)>> = ThemeState::toggle(theme_state.theme);

html! {
    euv_button {
        variant: EuvButtonVariant::Primary
        label: "Toggle Theme"
        onclick: toggle_theme
    }
}

检测系统主题

let theme: String = ThemeState::detect_system_theme();
// 返回 "light" 或 "dark"

// 监听系统主题变化
ThemeState::use_system_theme_change(theme_signal);

提示

use_theme_state 创建后的 theme 和 root_class 信号可以用于 html! 宏：root_class 用于 <html> 或 <body> 的 class 属性，theme 用于条件样式。系统主题变化通过 MediaQueryList 的 change 事件监听，当系统切换到暗色/亮色模式时自动更新。
