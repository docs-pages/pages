页头、信息行与 Logo

eastspire2026/7/19大约 1 分钟euvuirustwasmcomponent-library

euv_header

页面标题组件，展示 emoji 图标、标题和副标题，适合作为页面的起始部分。

Props

字段类型默认值说明
icon&'static str""Emoji 图标
title&'static str""标题文字
subtitle&'static str""副标题文字

基本用法

use euv_ui::*;

html! {
    euv_header {
        icon: "🚀"
        title: "Getting Started"
        subtitle: "Learn how to build amazing UIs with euv."
    }
}

euv_info

键值信息行组件，左侧 label 右侧 value（通过 children 传入），适合展示结构化信息。

Props

字段类型默认值说明
label&'static str""左侧标签

基本用法

use euv_ui::*;

let version: &str = "0.8.29";

html! {
    div {
        euv_info {
            label: "Version"
            version
        }
        euv_info {
            label: "Author"
            "root@ltpp.vip"
        }
    }
}

euv_logo

品牌 Logo 组件，显示带渐变背景的 "E" 字母，支持 Nav（导航栏）和 Fab（浮动按钮）两种变体。

Props

字段类型默认值说明
variantLogoButtonVariantNav显示变体
on_clickOption<Rc<dyn Fn(Event)>>None点击回调

LogoButtonVariant

变体说明
Nav32×32 内联按钮，用于导航侧边栏
Fab48×48 固定定位浮动按钮

基本用法

use euv_ui::*;

html! {
    euv_logo {
        variant: LogoButtonVariant::Nav
        on_click: move |_: Event| {
            // 导航到首页
        }
    }
}

提示

当 on_click 为 None 时，euv_logo 渲染为 <span> 元素，可嵌套在 <a> 标签内使用。
